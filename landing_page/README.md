- **`script.js`** - JavaScript功能文件，包含所有交互功能

## 快速开始

### 本地查看
1. 确保 `index.html`, `styles.css`, `script.js` 在同一文件夹中
2. 双击 `index.html` 在浏览器中打开
3. 无需服务器，可直接在浏览器中查看

### 打包发送
只需要发送这3个文件：
- `index.html`
- `styles.css`
- `script.js`

详细说明见 `PACKAGE_FILES.md`

## 主要功能

### 1. 动态标题
- H1 标题中的词（Baby, Brand, Character）自动循环切换
- 平滑的淡入淡出动画

### 2. 标签页切换
- Generate / Narrow Down 标签页切换
- Use Cases 左右分栏标签切换

### 3. FAQ 手风琴
- 3个FAQ问题可展开/折叠
- 平滑的动画效果

### 4. 响应式设计
- 移动端、平板、桌面端完美适配
- 触摸友好的交互设计

## 设计系统

所有设计元素遵循 `design_system.md` 中的规范：

- **颜色**: 温暖、明亮、科技向善的配色方案

- **间距**: 4px 基础单位的间距系统
- **圆角**: 8px-16px 的圆角系统
