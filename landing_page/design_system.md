# uniquenames.net 设计系统文档

**项目**: 落地页第一版设计规范  
**创建日期**: 2025-01-31  
**设计风格**: 温暖、明亮、美好、有安全感、带一点点科技向善

---

## 设计理念

### 核心关键词
- **温暖** (Warm): 柔和的色彩，友好的视觉语言
- **明亮** (Bright): 充足的留白，清晰的层次
- **美好** (Beautiful): 精致的细节，优雅的动效
- **安全感** (Secure): 稳定的布局，可信的视觉元素
- **科技向善** (Tech for Good): 现代但不冰冷，科技感但有人情味

## 动画和过渡

### 过渡时长
```
快速: 150ms (微交互)
标准: 200ms (默认)
慢速: 300ms (重要变化)
很慢: 500ms (页面过渡)
```

### 缓动函数
```
标准: cubic-bezier(0.4, 0, 0.2, 1) (Material Design)
进入: cubic-bezier(0.0, 0, 0.2, 1) (加速)
退出: cubic-bezier(0.4, 0, 1, 1) (减速)
弹跳: cubic-bezier(0.68, -0.55, 0.265, 1.55) (友好提示)
```

### 动画效果
```
淡入: opacity 0 → 1
上滑: translateY(10px) → 0 + opacity 0 → 1
缩放: scale(0.95) → 1
颜色过渡: 所有颜色变化使用过渡
```


## 图标和装饰

### 图标风格
```
风格: 线性图标 (Line Icons)
粗细: 1.5px - 2px
圆角: 轻微圆角端点
颜色: 继承文字颜色或使用强调色
尺寸: 16px, 20px, 24px
```

### 装饰元素
```
角度: 轻微倾斜 (2-5度)，增加动感但不失稳定
形状: 柔和曲线，避免尖锐角度
透明度: 使用 rgba 实现层次感
```

## 设计原则总结

1. **温暖优先**: 所有颜色和形状都避免冰冷和尖锐
2. **明亮清晰**: 充足的留白，清晰的层次
3. **精致细节**: 圆角、阴影、过渡都精心设计
4. **稳定可靠**: 布局稳定，视觉元素可信
5. **科技友好**: 现代但不冰冷，科技感但有人情味
